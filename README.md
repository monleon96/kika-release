# KIKA App Releases

This repository contains pre-built releases of the KIKA desktop application for nuclear data visualization and analysis.

## What is KIKA?

KIKA is a desktop application for visualizing and analyzing nuclear data files. It provides an intuitive interface for working with Monte Carlo simulation data, sensitivity analysis, and nuclear data processing.

## Download & Installation

### Latest Release

Go to the [Releases](../../releases) page to download the latest version for your platform:

- **Windows**: `kika-desktop-windows.exe`

### Installation Instructions

#### Windows
1. Download the `.exe` file
2. Run the installer and follow the setup wizard
3. Launch KIKA from the Start Menu or desktop shortcut
